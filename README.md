# amoghptl.github.io

Personal website for **Amogh Patil** — smart contract developer and Web3 founder — live at **https://amoghptl.github.io**.

Built on the [AcademicPages](https://github.com/academicpages/academicpages.github.io) Jekyll template (a fork of [Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes)) and hosted free on GitHub Pages.

## Where things live

| Content | File / directory |
|---|---|
| Site config & sidebar | `_config.yml` |
| Homepage | `_pages/about.md` |
| CV | `_pages/cv.md` |
| Navigation bar | `_data/navigation.yml` |
| Projects | `_portfolio/` |
| Blog posts | `_posts/` |
| Profile photo | `images/profile.jpg` (replace with your own) |

## Publishing

Pushing to the `main` branch triggers an automatic GitHub Pages build. No local toolchain is required to deploy.

To preview locally (optional), install Ruby + Bundler and run:

```bash
bundle install
bundle exec jekyll serve   # http://localhost:4000
```
